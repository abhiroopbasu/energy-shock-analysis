# Energy Shock Analysis: COVID & LPG Crisis in India

This project analyzes the impact of COVID-19 and energy supply disruptions on India's economy.

## Key Features
- Analysis of inflation and GDP trends
- Identification of economic shock periods
- Visualization of macroeconomic indicators

## Tools Used
- Python (pandas, matplotlib)

## Insights
- COVID caused a sharp decline in GDP growth
- Inflation increased during supply disruptions
- Economic recovery followed post-COVID period
